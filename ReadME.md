# Framewise

## Prerequisites

```text
Node JS - v18.15.0
PNPM - v8.4.0
```

## Setup

```bash
pnpm install
```

## Run locally

```bash
pnpm dev
```

## To start storybook

```bash
pnpm storybook
```

## To run e2e tests

```bash
pnpm playwright-ui-test
```

## To generate e2e testcode

```bash
pnpm playwright-codegen
```
